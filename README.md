## 📌 진행 방식 
1. 1기/2기의 깃허브 문제 순서 따라가며 진행( 1주에 1~2문제 풀이 목표)
2. 각 미팅 전까지 그 주에 해당하는 문제 직접 구현해보기
3. 완성된 부분까지 팀원들과 공유
4. 수상자 코드 분석해온 내용 발표
5. 진행 속도에 따라 문제 개수 조정
※서로 진행 상황 자유롭게 공유하기!!
